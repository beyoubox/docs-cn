哈希宝藏¶

“哈希宝藏”是基于币游链的一款大型去中心化塔防游戏。游戏挖矿和邀请收益完全采用去中心化链上分配，公正透明。

玩家通过在“哈希宝藏”上搭建防御塔、升级或打怪等方式，来获得BYC代币奖励。“哈希宝藏”里的宝石则和币游链平台的USDB稳定币1:1兑换，用于购买游戏中所需要消耗的物资等。

玩家花费的宝石越多，获得的BYC代币奖励也越多。

新手引导¶

当玩家首次登陆哈希宝藏时，即可免费获得含有20宝石新手礼包（价值20USDT）。此外，平台推出限时首充奖励——玩家首月首次充值，即可享受10%优惠。如：首充100宝石，即享110宝石。

玩家接收新手礼包后，平台将自动部署建造最低级别的防御塔——价值20宝石的弓箭塔，使玩家立刻享受防御塔收益。获得收益后，玩家可通过向平台充值宝石的方式，不断升级防御塔，每日收获更多的BYC，以求获得更高的回报率。

防御塔共有10个级别，玩家所拥有的防御塔数量越多，等级越高，越具有攻击力，抵御能力也越高，玩家因此获得的BYC代币收益也就越高。当玩家将防御塔打造到最高级别后，平台将开发“无尽宝藏”模式，进行无限升级。

在游戏过程中，防御塔可持续120天进行不间断的收益。但是，玩家需要每3天进行一次防御塔维护。每次维护需要消耗2颗宝石，以保证防御塔能每日正常收益。防御塔的维护费用会随着时间的增加，不断减少。当维护费用降为0时，防御塔将停止收益。

同时，玩家还可以通过“点击炼石”的方式，将已产生BYC收益进行复投，每次复投可享受平台5%额外奖励。

根据投资额度的不同，玩家的月收益率基本都保持在10%-20%的范围内。玩家可将在游戏中获得BYC代币收益，通过在交易所上售出获得利润，120天每日循环不停赚钱。

玩家需要注意的是，每次提现需要扣除5%手续费，作为平台的系统维护费用，且每次提现额度最低为10 BYC。

邀请收益¶

玩家可通过使用专有HID身份码邀请好友，当玩家投资额大于等于一级邀请人投入资金一半时，获得一级邀请人最低40%，最高100%的每日利息收益。

为了提高游戏社群的互动性，币游链开发团队特此开展“矿主联盟”计划。每位解锁白银矿工等级的玩家，即可获得600宝石的奖励。当玩家邀请好友达到条件时，自动加入矿主联盟。

其中，矿主分为「宝石矿主」、「白金矿主」、「黄金矿主」，其身份依靠贡献度提升。贡献度则可以通过邀请的好友及好友的团队充值数量计算，每100宝石 = 100USDT = 1 贡献度，矿主自身充值不会增加自己己的贡献度。

加入矿主联盟后，玩家可获得矿主身份及联盟新贡分红，还可以根据矿主身份获得BYC全球矿主分红奖励： 黄金矿主将获得其自身联盟中每周新增贡献度的7%作为矿主联盟收益（BYC）。 白金矿主将获得其自身联盟中每周新增贡献度的5%作为矿主联盟收益（BYC）。 宝石矿主将获得其自身联盟中每周新增贡献度的3%作为矿主联盟收益（BYC）。

为鼓励和感谢更多非矿主游戏玩家，“哈希宝藏”还将推出「质押挖矿」功能。平台会将每周入金USDT/ETH的10%放入质押分红池中, 按照玩家质押比例每周发放分红。

为进一步确保玩家的收益，我们将推出由智能合约自动执行的平台兑换功能。“哈希宝藏”游戏平台永久支持代币兑换，以创世价格为基准(BYC =0.05 USDT)的保底价格，永久支持玩家向游戏平台兑换其游戏收益所得。并且平台会自动收取玩家20%的收益，用作支付平台的矿工费用。
